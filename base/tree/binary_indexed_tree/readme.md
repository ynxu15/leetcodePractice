# 二维数组
差分定义
d[i][j] = nums[i][j] - nums[i-1][j] - nums[i][j-1] + nums[i-1][j-1]

如何根据差分数组获得原始数组
nums[i][j] = for ii in range(i+1) for jj in range(j+1) ans += d[ii][jj]

范围增加操作
[i,j] - [ii, jj] 增加v
那么，只需要 改动4个数
d[i,j] +v
d[i+1,jj] -v
d[ii,j+1] -v
d[ii+1,jj+1] +v
